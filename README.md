Tools & Technologies:- Power BI, Power Query, DAX, Data Modeling, Data Visualization, Forecasting
Project Overview:- Developed an interactive Power BI dashboard to analyze Mutual Funds data, providing Asset Classes (Equity, Debt, Cash), Market Capitalization, Credit Ratings (AAA/AA/A), AUM Analysis, Rolling Returns, Portfolio Allocation

Key Contributions & Insights:-
Designed and developed an end-to-end Power BI dashboard analyzing mutual fund performance, asset allocation, sector exposure, credit quality, and market capitalization trends.
Built a star schema data model integrating fact tables (Holdings, Sector Allocation, Returns) with dimension tables (Date, Scheme, Sector) to enable scalable analytics.
Created DAX measures for rolling returns (1M, 3M, 1Y, 3Y, 5Y, Since Inception), AUM tracking, credit quality distribution, and sector allocation analysis.
Implemented time-intelligence calculations and dynamic filtering using slicers and drill-through features.
Performed data validation, reconciliation, and transformation to ensure data integrity across multiple sources.
Optimized data model relationships and measures to improve report performance and usability.
Delivered interactive visualizations including sector trend analysis, fund comparison, asset class breakdown, and risk-return insights.
